# Simple Interest Calculator

This project is a simple interest calculator that calculates simple interest based on the following formula:

Simple Interest = (Principal × Rate × Time) / 100

## Inputs
- Principal: The initial amount of money
- Rate: The rate of interest per year
- Time: The time period in years

## Output
- Simple Interest: The calculated interest amount

## Usage
This calculator can be used to quickly determine the interest earned on an investment or loan using the simple interest formula.
